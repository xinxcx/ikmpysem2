1. Постановка задачи
Дана кольцевая строка цифр длины N (≤ 1000). Требуется найти три последовательных числа A, B, C в кольце таких, что A + B = C. Цифры используются ровно один раз в порядке следования.
Формат ввода: Файл input.txt с одной строкой цифр (пример: "01902021").
Формат вывода: Файл output.txt с тождеством A+B=C (пример: "190+20=210") или "No" если решения нет.

2. Структура данных и обоснование
Выбранная структура: Стандартные списки Python (list) и процедурный подход.
Почему:

Задача имеет алгоритмическую, а не объектно-ориентированную природу

Основные операции — перебор и арифметические вычисления

Минимизация накладных расходов для работы с большими N (до 1000)

Прямой доступ к элементам по индексу через O(1)

3. Алгоритм решения
Ключевые шаги:

Преобразование строки в список цифр (например, "123" → [1,2,3])

Проверка минимальной длины: Если N < 3 → вывод "No"

Двойной перебор:

Для каждого старта A (0 ≤ start < N)

Для каждой длины A (1 ≤ a_len ≤ N-2)

Вычисление кандидатов для длины B:

cand1 = n - 2 * a_len      # Вариант: a_len >= b_len
cand2 = n - 2 * a_len - 1  # Поправка для нечётных
cand3 = (n - a_len) // 2   # Вариант: b_len >= a_len

Проверка чисел:

Нет ведущих нулей: 

if a_len > 1 and digits[start] == 0: 
	skip

Кольцевая арифметика: idx = (start + i) % n

Проверка условия: A + B == C

Запись результата или вывод "No"

Сложность: O(N²) → максимум 10⁶ операций при N=1000 (приемлемо).

4. Обработка исключений
Критические места:

Пустой ввод:

if len(s) == 0:
    with open("output.txt", "w") as out:
        out.write("No")

Ведущие нули:

python
if a_len > 1 and digits[start] == 0: 
    continue

Некорректные длины:

python
if c_len < 1: 
    continue
